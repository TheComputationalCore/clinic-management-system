# Security Policy

## Supported Versions
The project is actively maintained on the **main** branch.  
Security patches will be applied to the latest stable release.

## Reporting a Vulnerability
If you discover a security vulnerability, please follow these steps:

1. **Do NOT open a public GitHub issue!**
2. Email the maintainer privately:

**Dinesh Chandra — TheComputationalCore**  
GitHub: https://github.com/TheComputationalCore

3. Provide:
   - A clear description of the vulnerability
   - Steps to reproduce (if possible)
   - Potential impact
   - Any suggestions for remediation

You will receive a confirmation within 48–72 hours.

## Responsible Disclosure
Please allow reasonable time for the issue to be resolved before public disclosure.
